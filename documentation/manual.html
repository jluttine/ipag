Instructions for compiling and use

Program is tested on Linux and Windows. It requires libSDL and optionally SDL_mixer for sound. It can be compiled with make command. Make sure to unpack resource files. There should be a subdirectory called data.

Sounds are disabled by default. If SDL_mixer is installed properly, you just need to add linker flag -lSDL_mixer and uncomment #define SOUND in sounds.h.

SDL_mixer at HUT

This is a bit tricky and only necessary if you want to hear sound at HUT computers (there are no speakers, by the way).

Uncomment following lines in Sounds.h
#define SOUND
#include "SDL_mixer.h"

Comment following line out in Sounds.h
#include <SDL/SDL_mixer.h>

You have to set enviroment variable LD_LIBRARY_PATH by typing
setenv LD_LIBRARY_PATH "/p/contrib/lib"
in command prompt.

Compile program with following command:
g++ -g -o ipag *.cpp `sdl-config --cflags --libs` -lSDL -lSDL_mixer -L/p/contrib/lib

Features

  • Multiplayer game on a single computer
  • Different AI players
  • Lots of different weapons
  • A shop where user can buy weapons
  • Continuous world and scrolling
  • Rotating planets

Playing

Start a new game

First you must add players. Type the name of the new player and select it's type by clicking the arrows. Player can be human or one of three different AIs. Add the new player by clicking Add player button or by pressing return. You can add up to eight players. If you don't type a name, a default name is used.

In the next window you can adjust parameters of the world. Choose the number of planets and keep clicking Create button until you see a world that pleases you. Click Finish to move on.

Next you will have to choose location for your cannon. Do this by clicking a spot on the screen. You can't place the cannon too close to an existing cannon. When all players have selected their locations, click Finish.

In shop window you can buy weapons an shields. Click a weapon's name to see a description.

In the game you aim with left and right arrow key, set power with up and down arrow keys and shoot with space bar. You can fine-tune by keeping CTRL key down while adujusting the aiming. On the lower right corner you can select your weapon and adjust parametes by clicking the arrows. You can relocate your cannon by clicking Move button and selecting a new spot on the game area, but it will cost you your turn.

Game ends after all cannons but one are destroyed. A score window appears and after that a new round starts.

You can usually quit by pressing esc key a few times. In game window this is, however, disabled to avoid accidents.
